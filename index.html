zmon.it: fast open scalable monitoring platform

zmon.it is

  • fast - built on top of ZeroMQ and Malamute broker
  • open - core, basic components and web interface under opensource licenses
  • scalable - can support more devices and events than any other solution
  • monitoring - can talk via various protocols (SNMP, IPMI, ...)
  • platform - via defined protocols it can be easily adapted to your needs
